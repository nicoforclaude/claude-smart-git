# Git Menu

Available git commands:

## /git:status
Show git status — current branch, staged, unstaged, and untracked files

## /git:to-main
Switch to main branch, with safety check for pending changes

## /git:branch:create
Smart branch creation — finds naming convention, infers topic from session

## /git:branch:cleanup
Delete the current branch once fully merged into main — lands on main when done

## /git:worktree:show
List all worktrees with their branch and pending changes status

## /git:pull-main
Fetch and merge origin/main into the current branch

## /git:sync
Diagnose and sync branches downstream (deploy→main→working)

## /git:startup
Quick git status check at session start

## /git:push
Push commits to remote repository

## /git:commit:fast
Quick single commit — no linting, no change analysis. Pass message as argument or let it infer one.

## /git:pr:create
Create a PR — infers title and body from session context and commits

## /git:pr:fix-ci
Fetch failing CI logs, reproduce locally, fix, and verify — reads `.claude/ci-map.md` for keyword→check mapping

## /git:pr:code-review:resolve-one-by-one
Read PR review comments, triage by severity (Critical/LFH/Normal), fix one at a time

---

**Tip**: For commits, use `/git:commit` (full) or `/git:commit:fast` (quick)
